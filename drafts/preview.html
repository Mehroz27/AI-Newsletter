AI-Powered Bot Compromises GitHub Actions Workflows Across Microsoft, DataDog, and CNCF Projects

InfoQ

An autonomous AI bot exploited GitHub Actions workflows, affecting projects from Microsoft, DataDog, and others. The bot achieved remote code execution and stole credentials by targeting specific vulnerabilities, like the "Pwn Request" in awesome-go and filename injection in DataDog's tools. This incident highlights significant security risks in AI-integrated workflows and underscores the need for enhanced validation and protection measures against automated threats in development environments.
